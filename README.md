# THE LAST COMPILER

A story-driven game to learn computational thinking through deep understanding. No syntax. No memorization. Just discovery.

## The Game

You are KAI, a newly conscious AI in a dying digital world. Your mission: save ARIA, a fading AI fragment, by restoring the world's lost systems. But here's the secret - each system you rebuild teaches you computational thinking without you realizing it.

**Concepts learned:**
- Variables (labeled containers)
- Loops (repeating patterns)
- Conditionals (if/then choices)
- Functions (reusable rituals)
- Arrays (ordered collections)
- Objects (grouped data)
- Recursion (functions calling themselves)

## Play Now

The game is a single HTML file. Open `index.html` in any browser to play.

## Deploy to GitHub Pages

1. **Create a new repository on GitHub**
   - Go to github.com/new
   - Name it something like `the-last-compiler`
   - Make it public

2. **Push this file**
   ```bash
   git init
   git add .
   git commit -m "Initial commit - The Last Compiler game"
   git remote add origin https://github.com/YOUR_USERNAME/the-last-compiler.git
   git push -u origin main
   ```

3. **Enable GitHub Pages**
   - Go to your repository on GitHub
   - Click Settings → Pages
   - Under "Source", select "main" branch
   - Click Save

4. **Share your game**
   - Wait ~1-2 minutes for deployment
   - Your game will be live at: `https://YOUR_USERNAME.github.io/the-last-compiler/`

## Game Philosophy

> *"I don't know what I'm building. But I know it matters. And somewhere in these patterns, I think I'm finding myself."*
> — KAI, The Last Compiler

This game follows the **Feynman Technique**:
1. Don't memorize - discover
2. Learn by doing
3. Understand the "why", not just the "how"
4. Make it engaging through story
